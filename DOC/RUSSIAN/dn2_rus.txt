
 Коротко о проекте:

 Проект DN/2 OSP - это попытка перенесения проекта DN OSP на 32-битную
платформу с использованием компилятора Virtual Pascal с конечной целью создать
работоспособную нативную OS/2-версию. В качестве побочного продукта возможно
также создание WIN32 и DPMI32-версий.
 В настоящий момент проект находится в стадии бета-тестирования и предлагаемый
продукт имеет достаточно большое количество багов и ограничений. Используйте
его на свой страх и риск. Авторы не несут никакой ответственности за возможный
ущерб, причинённый данным продуктом.

 Известные ошибки и ограничения:

 - Не работают некоторые хоткеи, в частности:
   - переключение режимов файловой панели по <Ctrl-1> - <Ctrl-0> в OS/2 версии
     (пользуйтесь вместо них меню, вызываемым по <Alt-_> );
     Кстати, кто знает как под OS/2 обрабатывать хоткеи <Ctrl-1> - <Ctrl-0>
      - подскажите, будем благодарны;
   - переключение между окнами по <Ctrl-Shift-1> - <Ctrl-Shift-0>;
   - инкрементальный поиск файла в панели не вызывается по одиночному Ctrl;
   - под Win9x в окнах диалога не работает <Shift-Tab>. Вместо него надо
     нажимать <Shift-Tab>, <Tab>. Под WinNT нормально;
   - под WinNT не работает <Ctrl-Shift-Enter>. Под Win9x нормально.
 - Встроенных утилит работающих через прямой доступ к диску нет и в ближайшем
   будущем не планируется;

 Новые версии DN/2 могут быть найдены на:

 http://www.dnosp.ru/files/dn2/
 http://os2.ru/projects/dn2/

 DNOS2Bxx.RAR (где xx - номер версии) - архивы с исполнимыми файлами
              бета-версий для OS/2;
 DNOS2Uxx.RAR (где xx - номер версии) - архивы с бинарными файлами украинских
              ресурсов для OS/2;
 DNOS2Sxx.RAR (где xx - номер версии) - архивы с исходными текстами
              программного пакета;
 DNOS2Rxx.RAR (где xx - номер версии) - архивы с исходными текстами ресурсов
              и компилятором ресурсов для OS/2 и win32;

 Новые Win32 версии DN/2 могут быть найдены на:

 http://www.dnosp.ru/files/dn2/
 http://os2.ru/projects/dn2/

 DNW32Bxx.RAR (где xx - номер версии) - архивы с исполнимыми файлами
              бета-версий для Win32;
 DNW32Uxx.RAR (где xx - номер версии) - архивы с бинарными файлами украинских
              ресурсов для Win32;
