
==============================================================================
                           название и версия патча
              A Contribute to Dos Navigator Open Source Project

==============================================================================

                Author(s): имена и координаты авторов

                 Based on: название и версия патча, послужившего основой,
                           или слово Scratch (с нуля)

 Applicable to DN version: первоначально 1.51 (=1.51.00/DOS), далее номера
                           релизов DN Open Source будут прирастать цифрами
                           после второй точки, а через дробь будет указываться
                           операционная система (DOS, W32, DPMI, OS2, Linux,
                           ..., что еще на горизонте появится?) или звездочка,
                           если патч не зависит от операционной системы.

              Description: описание патча - что он добавляет в DN, какие его
                           ошибки исправляет, какие добавляются/изменяются
                           команды меню, диалоговые окна, горячие клавиши;
                           желательно не связанные между собой изменения не
                           включать в один патч.

    Source files modified: список файлов исходников, подвергшихся изменению

       Source files added: список добавленных файлов исходников, если есть

  Affected DN language(s): список языков, для которых патч содержит обновления
                           ресурсов и/или справки, или звездочка, если патч не
                           зависит от языка (скажем, изменяет логику работы с
                           архивами)

  Tools required to apply: утилиты (с указанием минимальной версии),
                           необходимые для применения патча к исходникам;
                           скорее всего это утилита patch, но могут быть,
                           например, какие-нибудь применители бинарных патчей

     Tools used to create: утилиты (с указанием версии), использованные для
                           создания патча; скорее всего это утилита diff, но
                           могут быть какие-нибудь утилиты для создания
                           бинарных патчей

  Known incompatibilities: список других патчей (с указанием версий), особых
                           случаев применения DN и/или других программных
                           средств, используемых вместе с DN, с которыми, как
                           известно автору, данный патч несовместим

               Known bugs: известные ошибки в этом патче, которые автор не
                           исправил на момент его выпуска

                    To do: желательные добавления или изменения патча

==============================================================================
